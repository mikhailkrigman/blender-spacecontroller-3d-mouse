# SpaceController 3D Mouse for Blender 4.x

Unofficial Blender 4.x add-on that lets you use **SpaceController** 3D mice for viewport navigation and camera control.

> ⚠️ This project is **not** affiliated with or endorsed by SpaceControl GmbH.  
> “SpaceController” is a trademark of SpaceControl GmbH.

---

## Features

- Smooth **6-DOF viewport navigation** using a SpaceController 3D mouse
- Works with **Blender 4.x** (and should be compatible with most 3.x versions)
- Simple Settings-UI in the **N-panel** of the 3D View

---

## Requirements

- **Blender:** 4.0+ (tested with 4.4 / 4.5)
- **OS:** Windows 10/11 (other platforms currently not supported)
- **Hardware:** SpaceController 3D mouse
- **Drivers:** Official SpaceControl driver / software installed  
  (so that the device is visible to the system and/or their DLL is available)

---

## Installation

1. **Download the add-on**
   - Either clone the repo:

     ```bash
     git clone https://github.com/<your-username>/blender-spacecontroller-3d-mouse.git
     ```

   - Or download it as a ZIP from GitHub.

2. **Install in Blender (recommended way)**
   1. In Blender, go to:  
      `Edit → Preferences… → Add-ons → Install…`
   2. Select the ZIP file.
   3. Enable the add-on: search for **“SpaceController 3D Mouse Addon”**.
